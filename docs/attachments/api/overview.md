# ファイル添付API 概要

このAPIは、学習記録に対してファイルを添付・取得・削除するためのものです。

## 主な機能

- 学習記録へのファイルアップロード
- 添付ファイル一覧の取得
- ファイルのダウンロード
- 添付ファイルの削除

## 想定ユースケース

- 勉強した内容のPDFや画像を学習記録に紐付けて管理する
- 手書きメモのスキャン・スクリーンショットを学習記録と一緒に残す
- 参考資料（Excel・テキスト等）を後から参照できるよう保存する

## 共通仕様

| 項目 | 内容 |
|---|---|
| ベース URL | `/api` |
| 認証 | `Authorization: Bearer {JWT}` ヘッダー必須（全エンドポイント） |
| アップロード時の Content-Type | `multipart/form-data` |
| 一覧取得・削除時の Content-Type | `application/json` |
| レスポンス形式 | JSON（ダウンロードAPIを除く） |
| 文字コード | UTF-8 |

## 制約

| 項目 | 制限 | 備考 |
|---|---|---|
| 1ファイルの最大サイズ | 10MB | 大きすぎるファイルはサーバーのメモリを圧迫するため上限を設ける |
| 1学習記録あたりの最大添付数 | 10ファイル | ストレージの無制限消費を防ぐため。学習記録1件に対して現実的な上限として設定 |
| 対応ファイル形式 | 制限なし（MIMEタイプをDBに記録） | 学習用途はPDF・画像・Excelなど多様なため形式は絞らない。MIMEタイプをDBに保存することでダウンロード時に正しい `Content-Type` を返せるようにする |
