# 学習記録管理API 概要

このAPIは、ユーザーの学習活動の記録・取得・更新・削除を行うためのものです。

## 主な機能
- 学習記録の新規作成
- 学習記録の一覧取得
- 学習記録の詳細取得
- 学習記録の更新
- 学習記録の削除
- タグ付与・検索

## 想定ユースケース
- 日々の学習内容を記録し、振り返りや分析に活用する
- タグで学習内容を分類し、検索や集計を容易にする
- ユーザーごとに独自タグを作成し、柔軟な管理を実現

## 想定利用者
- Webエンジニア、ITエンジニア、転職活動中エンジニア

## 共通仕様

| 項目 | 内容 |
|---|---|
| ベース URL | `/api` |
| バージョニング | 現時点ではバージョンなし。将来的に `/api/v2` へ移行する場合は全エンドポイントに `/v2` を追加 |
| 認証 | `Authorization: Bearer {JWT}` ヘッダー必須（認証不要のエンドポイントはない） |
| Content-Type | `application/json` |
| レスポンス形式 | JSON |
| 文字コード | UTF-8 |
