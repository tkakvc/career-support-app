# エラー設計

## エラーコード一覧

| HTTPステータス | 内容 | 発生ケース |
|---|---|---|
| 400 | バリデーションエラー | 必須項目の欠落、文字数超過、値の範囲外 |
| 401 | 認証エラー | JWT トークンが不正・期限切れ・未送信 |
| 403 | 権限エラー | 他ユーザーのリソースへのアクセス、default タグの編集・削除 |
| 404 | リソース未発見 | 存在しない ID の指定、存在しない tagId の指定 |
| 409 | 重複エラー | 同ユーザーで同名タグをすでに保有している |
| 500 | サーバーエラー | 予期しないサーバー内部エラー |

---

## エラーレスポンス形式

```json
{
  "error": "エラーメッセージ",
  "status": 400
}
```

## エンドポイント別エラーケース

### POST /api/learning-records
| 条件 | ステータス |
|---|---|
| `content` が空 | 400 |
| `content` が2000文字超 | 400 |
| `duration` が 0 以下または 1440 超 | 400 |
| `date` が未来日 | 400 |
| 存在しない `tagId` を指定 | 404 |
| JWT 不正 | 401 |

### GET /api/learning-records/{id}
| 条件 | ステータス |
|---|---|
| 存在しないIDを指定 | 404 |
| 他ユーザーのIDを指定 | 403 |
| JWT 不正 | 401 |

### PUT /api/learning-records/{id}
| 条件 | ステータス |
|---|---|
| 存在しないIDを指定 | 404 |
| 他ユーザーのIDを指定 | 403 |
| バリデーションエラー（POSTと同様） | 400 |
| JWT 不正 | 401 |

### DELETE /api/learning-records/{id}
| 条件 | ステータス |
|---|---|
| 存在しないIDを指定 | 404 |
| 他ユーザーのIDを指定 | 403 |
| JWT 不正 | 401 |

### POST /api/tags
| 条件 | ステータス |
|---|---|
| `name` が空 | 400 |
| `name` が50文字超 | 400 |
| 同ユーザーで同名タグが存在する | 409 |
| JWT 不正 | 401 |

### PUT /api/tags/{id}
| 条件 | ステータス |
|---|---|
| 存在しないIDを指定 | 404 |
| default タグを指定 | 403 |
| 他ユーザーが作成したタグを指定 | 403 |
| 同ユーザーで同名タグが存在する | 409 |
| JWT 不正 | 401 |

### DELETE /api/tags/{id}
| 条件 | ステータス |
|---|---|
| 存在しないIDを指定 | 404 |
| default タグを指定 | 403 |
| 他ユーザーが作成したタグを指定 | 403 |
| JWT 不正 | 401 |

## 備考

- タグを削除しても、紐づく学習記録は削除されない。タグとの紐付けのみ解除される。
