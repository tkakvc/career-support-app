# エラー設計

## エラーコード一覧

| HTTPステータス | 内容 | 発生ケース |
|---|---|---|
| 400 | バリデーションエラー | ファイルサイズ超過、ファイル未添付、添付数上限超過 |
| 401 | 認証エラー | JWT トークンが不正・期限切れ・未送信 |
| 403 | 権限エラー | 他ユーザーの学習記録・添付ファイルへのアクセス |
| 404 | リソース未発見 | 存在しない学習記録ID・添付ファイルIDの指定 |
| 500 | サーバーエラー | ストレージへの保存失敗、予期しないサーバー内部エラー |

---

## エラーレスポンス形式

```json
{
  "error": "エラーメッセージ",
  "status": 400
}
```

---

## エンドポイント別エラーケース

### GET /api/learning-records/{id}/attachments

| 条件 | ステータス |
|---|---|
| 存在しない学習記録IDを指定 | 404 |
| 他ユーザーの学習記録IDを指定 | 403 |
| JWT 不正 | 401 |

### POST /api/learning-records/{id}/attachments

| 条件 | ステータス |
|---|---|
| `file` が未添付 | 400 |
| ファイルサイズが10MBを超える | 400 |
| 添付ファイル数がすでに10件に達している | 400 |
| 存在しない学習記録IDを指定 | 404 |
| 他ユーザーの学習記録IDを指定 | 403 |
| ストレージへの保存に失敗 | 500 |
| JWT 不正 | 401 |

### GET /api/learning-records/{id}/attachments/{attachmentId}/download

| 条件 | ステータス |
|---|---|
| 存在しない学習記録IDを指定 | 404 |
| 存在しない添付ファイルIDを指定 | 404 |
| 他ユーザーの学習記録IDを指定 | 403 |
| ストレージ上のファイルが見つからない | 500 |
| JWT 不正 | 401 |

### DELETE /api/learning-records/{id}/attachments/{attachmentId}

| 条件 | ステータス |
|---|---|
| 存在しない学習記録IDを指定 | 404 |
| 存在しない添付ファイルIDを指定 | 404 |
| 他ユーザーの学習記録IDを指定 | 403 |
| JWT 不正 | 401 |

---

## 備考

- 学習記録を削除したとき、紐づく添付ファイルはストレージとDBの両方から削除する（カスケード削除）
