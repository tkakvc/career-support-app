# エラー設計

## エラーコード一覧

| HTTPステータス | 内容 | 発生ケース |
|---|---|---|
| 400 | バリデーションエラー | goal が未入力・200文字超過 |
| 401 | 認証エラー | JWT不正・期限切れ |
| 429 | レート制限 | OpenAI APIのレート制限に達した |
| 500 | サーバーエラー | OpenAI APIの呼び出し失敗・タイムアウト・JSONパース失敗 |

## 429 の扱い

OpenAI API が `429 Too Many Requests` を返した場合、そのまま 429 をクライアントに返す。
リトライはしない（無限リトライによるコスト増加を防ぐため）。

## OpenAI APIのJSON出力が壊れていた場合

OpenAI は確率的に不正なJSONを返すことがある。
パースに失敗した場合は 500 を返し、ユーザーには「もう一度試してください」を促す。
