# リクエスト・レスポンス定義

---

## POST /api/ai/suggest　学習提案生成API

### 概要

ログインユーザーの直近30件の学習記録とタグ情報をOpenAIに送り、次に学ぶべき技術を3つ提案する。

リクエストボディは不要。ユーザーIDはJWTから取得する。

### リクエスト

ボディなし。

### OpenAIに送るプロンプト（内部処理）

```
あなたはエンジニアのキャリア支援AIです。
以下の学習記録をもとに、このエンジニアが次に学ぶべき技術を3つ提案してください。

【直近の学習記録】
{学習日・内容・タグの一覧}

以下のJSON形式で返してください。説明文は日本語で。
{
  "suggestions": [
    { "title": "技術名", "reason": "提案理由" },
    { "title": "技術名", "reason": "提案理由" },
    { "title": "技術名", "reason": "提案理由" }
  ]
}
```

### レスポンス（200 OK）

```json
{
  "suggestions": [
    {
      "title": "Docker",
      "reason": "Spring Bootの学習が進んでいるため、ローカル環境の構築やデプロイに必要なDockerを学ぶと実践的なスキルにつながります"
    },
    {
      "title": "GitHub Actions",
      "reason": "コードの品質管理のためCI/CDパイプラインの構築を学ぶと、チーム開発での貢献度が上がります"
    },
    {
      "title": "Redis",
      "reason": "APIのパフォーマンス改善としてキャッシュの概念を学ぶ段階に来ています"
    }
  ]
}
```

## POST /api/ai/decompose　タスク分解API

### 概要

ユーザーが入力した目標（例：「JWTログインを実装したい」）を、具体的な実装ステップに分解する。

### リクエスト

```json
{
  "goal": "JWTログインを実装したい"
}
```

| フィールド | 型 | 必須 | 制約 |
|---|---|---|---|
| goal | string | ○ | 1文字以上・200文字以内 |

### OpenAIに送るプロンプト（内部処理）

```
以下の目標を、エンジニアが実装できる具体的なタスクに分解してください。

目標: {goal}

以下のJSON形式で返してください。タスクは実装順に並べること。
{
  "tasks": [
    "タスク1",
    "タスク2",
    ...
  ]
}
```

### レスポンス（200 OK）

```json
{
  "tasks": [
    "User エンティティと UserRepository を作成する",
    "BCrypt でパスワードをハッシュ化する AuthService を実装する",
    "JWT を生成・検証する JwtTokenProvider を実装する",
    "リクエストごとに JWT を検証する JwtAuthenticationFilter を実装する",
    "Spring Security の設定（SecurityConfig）でフィルターを登録する",
    "ログインエンドポイント（POST /auth/login）を実装する"
  ]
}
```
